# Contributing

Thank you for helping improve the Bougie Bams Knowledge Base.

## Workflow

1. Choose the most specific folder for the topic.
2. Start from a template in `templates/`.
3. Fill every frontmatter field.
4. Keep the article focused on one question, concept, or workflow.
5. Add related document ids.
6. Run the validation and export scripts.

## Style

- Use plain, friendly teaching language.
- Keep sections short.
- Use examples that help a beginner understand the concept.
- Avoid unexplained jargon.
- Link related concepts instead of repeating long explanations.

## Copyright

Do not copy official NMJL rulebook text, annual card text, or specific annual card hands. Use original explanations. For official interpretation, direct players to consult the current official NMJL card or rulebook.
