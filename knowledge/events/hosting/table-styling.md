# Table Styling at Bougie Bams Events

## Summary

Every Bougie Bams event table is intentionally styled. Guests can choose from a library of more than 40 premium mats, tile sets, and racks so each table looks and feels the way they want.

## Details

The styling library pairs mats, tile sets, and racks into coordinated tablescapes. Linens, lighting, and accessories are chosen with the same intention — the atmosphere at the table is treated as equal in importance to the game itself.

At Style-Your-Own events, guests build their own table combination from the library, making the styling part of the entertainment.

For hosts and internal staff, the practical checklist per table of four: one mat sized to the venue table, one complete American mahjong set with jokers, four racks with pushers, and coordinated linens and accents.

## Examples

- A guest who loves a mat used at their event table can be pointed to the same or similar design in the web store.
- A venue with smaller cocktail tables calls for 31 inch mats rather than 33 inch.

## Common Mistakes

- Treating styling as decoration only; matched, complete equipment per table is what makes play smooth.
- Forgetting spare jokers and blanks in event kits.
