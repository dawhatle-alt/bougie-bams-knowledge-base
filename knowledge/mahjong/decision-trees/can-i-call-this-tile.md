# Can I Call This Tile?

## Summary

Use this decision tree to decide whether a discard can be claimed.

## Details

Decision trees are teaching aids. They do not replace the current official NMJL card or rulebook in formal play.

1. Does the discard complete a specific eligible group or Mahjong?
2. Does your intended hand allow this type of call?
3. Can you expose the required tiles clearly and immediately?
4. If official interpretation is uncertain, pause and consult the current rules.

## Examples

- Use this at the table before moving tiles.
- Use it in lessons to teach the reasoning sequence.

## Common Mistakes

- Skipping the validity check.
- Using the tree as a substitute for official rules in formal play.
