# Is This Hand Dead?

## Summary

Use this decision tree to slow down a possible dead-hand question.

## Details

Decision trees are teaching aids. They do not replace the current official NMJL card or rulebook in formal play.

1. Is there a clear rule issue or impossible exposure?
2. Has play continued after the possible mistake?
3. Can the table verify the issue with official materials?
4. If the answer is uncertain, stop tile movement and ask the host or official reference.

## Examples

- Use this at the table before moving tiles.
- Use it in lessons to teach the reasoning sequence.

## Common Mistakes

- Skipping the validity check.
- Using the tree as a substitute for official rules in formal play.
