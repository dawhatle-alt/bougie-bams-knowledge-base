# Host Guidelines

## Summary

A host creates the conditions for smooth, welcoming play.

## Details

Hosts should prepare equipment, explain table expectations, support new players, and keep the pace comfortable. For paid or formal events, document policies in the events and operations folders.

## Examples

- A host can place quick-reference materials near beginner tables.
- A host can pair newer players with patient instructors.

## Common Mistakes

- Overloading beginners with advanced corrections.
- Failing to explain house expectations before play begins.
