# Table Talk

## Summary

Table talk should support clarity without revealing private strategy or pressuring players.

## Details

Friendly conversation is part of casual Mahjong, but game-related comments can influence decisions. Groups should agree on how much coaching is appropriate.

## Examples

- A teacher-led table may allow questions after each turn.
- A social table may pause strategy talk during active play.

## Common Mistakes

- Hinting what another player should discard.
- Commenting on a player's likely hand in a way that changes the game.
