# Handling Disputes

## Summary

Disputes should be handled calmly, with respect for both learning and official references.

## Details

When players disagree, stop movement of tiles, restate the question, and consult the current official materials for formal interpretation. In casual settings, the host can decide how to proceed after everyone is heard.

## Examples

- A table can pause before moving a disputed discard.
- A host can separate a teaching explanation from an official ruling.

## Common Mistakes

- Arguing from memory when a reference is available.
- Making a player feel unwelcome for asking a rule question.
