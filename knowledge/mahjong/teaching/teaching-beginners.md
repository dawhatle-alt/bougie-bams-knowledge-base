# Teaching Beginners

## Summary

Beginner teaching works best when players learn the flow before advanced exceptions.

## Details

Start with tile recognition, table setup, turn rhythm, and simple decision language. Add calls, jokers, and strategy after the learner can follow a basic turn.

## Examples

- Ask beginners to identify suit and number before discussing hand selection.
- Run a face-up practice round to show flow.

## Common Mistakes

- Teaching exceptions before the core loop.
- Correcting so much that the learner stops experimenting.
