# First Lesson Plan

## Summary

A first lesson should build confidence with tiles, table flow, and simple choices.

## Details

A practical first lesson can cover equipment, tile families, seating, a slow demonstration turn, and one guided practice hand. Save advanced scoring and formal edge cases for later.

## Examples

- Start with a tile-sorting activity.
- End with each learner explaining one turn in their own words.

## Common Mistakes

- Trying to finish every rule in one class.
- Letting experienced players take over beginner decisions.
