# Dealing Tiles

## Summary

Dealing gives each player the starting tiles for the hand.

## Details

Dealing should be orderly and consistent so players begin with the correct number of tiles. For official settings, players should confirm the required procedure with the current NMJL reference.

## Examples

- A teacher can count aloud during a beginner lesson.
- Players can keep tiles concealed while organizing them on the rack.

## Common Mistakes

- Exposing tiles while dealing.
- Starting strategic discussion before everyone has checked their starting hand.
