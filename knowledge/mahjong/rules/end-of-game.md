# End Of Game

## Summary

A game ends when Mahjong is declared or when play reaches an agreed ending condition.

## Details

The end of a hand should include a clear stop, review if appropriate, and reset for the next hand. Scoring and formal outcomes should follow the current official card and rulebook.

## Examples

- A learning table can review the winning pattern after the hand ends.
- If no one wins, the table can reset according to its agreed rules.

## Common Mistakes

- Continuing to draw after the hand has ended.
- Debating scoring without the current official reference.
