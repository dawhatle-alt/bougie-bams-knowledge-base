# Optional Charleston

## Summary

The optional Charleston is an additional passing opportunity used only when permitted by the applicable rules.

## Details

Players should understand that optional means conditional, not automatic. In casual learning games, the group should state expectations before beginning.

## Examples

- A player can ask whether the table is using the optional pass before moving tiles.
- An instructor can explain the difference between required and optional phases.

## Common Mistakes

- Assuming every table handles optional passing the same way.
- Pressuring another player into a choice.
