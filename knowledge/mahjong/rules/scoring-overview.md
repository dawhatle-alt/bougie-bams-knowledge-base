# Scoring and Payouts Overview

## Summary

In American Mahjong, the winner collects a base value set by the current card, adjusted by how the win happened: the discarder of a winning tile pays double while the others pay single, a self-picked win is paid double by everyone, and a jokerless hand doubles the payout again.

## Details

Base value: every hand on the current card carries a printed value that reflects its difficulty. That value is the starting point for all payouts. This article describes the general structure only — consult the current official card for values.

Winning on a discard: when a player wins on another player's discard, the discarder pays the winner double the base value; the other two players each pay the single base value.

Winning by self-pick: when a player draws their own winning tile from the wall, all three opponents pay double the base value.

Jokerless bonus: completing a hand with no jokers doubles the payout from all players. This bonus does not apply to Singles and Pairs hands, which cannot use jokers anyway and carry higher printed values instead.

Stacking: a self-picked, jokerless win compounds both bonuses — each opponent pays four times the base value, the standard maximum multiplier.

Wall games: if the wall runs out with no winner, no money changes hands in most social play; the deal rotates and a new hand begins. Some groups use a kitty that rolls to the next winner, and some tournament formats award small bonuses to players whose hands are still viable.

## Examples

- A hand won on a discard: the discarder pays 2x the card value, the other two pay 1x each.
- A jokerless self-pick: all three opponents pay 4x the card value.

## Common Mistakes

- Applying the jokerless double to Singles and Pairs hands.
- Forgetting that dead-hand players still owe the winner at the end of the game.
