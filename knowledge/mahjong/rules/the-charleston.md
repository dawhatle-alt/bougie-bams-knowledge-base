# The Charleston

## Summary

The Charleston is a structured passing phase used before regular turns begin in American Mahjong.

## Details

Players use the Charleston to improve hand direction by passing unwanted tiles in a set pattern. This article explains the teaching concept only; players should consult the current official card or rulebook for official procedure.

## Examples

- A beginner can think of the Charleston as a chance to narrow options before the first draw.
- An instructor can ask players to name why they are passing each tile.

## Common Mistakes

- Passing tiles without a plan.
- Passing a tile that strongly supports a likely hand direction.
