# Setup Overview

## Summary

Setup prepares the table, equipment, seating, and reference materials before play begins.

## Details

A calm setup makes the rest of the game easier. Players should confirm the tile set is complete, agree on table expectations, and keep the current official reference available when using American Mahjong rules.

## Examples

- Place racks and dice before mixing tiles.
- Confirm that new players can see the discard area clearly.

## Common Mistakes

- Starting before everyone understands the table layout.
- Relying on memory when an official reference is needed.
