# Dead Hands

## Summary

A dead hand is a hand that can no longer legally continue because of a rules problem or impossible condition.

## Details

Dead-hand situations can be uncomfortable, so tables should handle them calmly and refer to the current official rules when needed. This article gives learning-oriented examples rather than formal rulings.

## Examples

- A hand may be questioned if an exposure cannot match any valid direction.
- A player may need help understanding why a table believes the hand cannot continue.

## Common Mistakes

- Using dead-hand calls casually or punitively.
- Failing to consult official rules for formal play.
