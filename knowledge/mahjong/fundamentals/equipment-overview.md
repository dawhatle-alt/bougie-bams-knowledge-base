# Equipment Overview

## Summary

A Mahjong table needs tiles, racks or trays, a playing surface, dice, and a current rule reference when using a card-based style.

## Details

Good equipment makes play smoother because everyone can see discards, organize tiles, and keep turns moving. Bougie Bams product guidance can later live in the product folders while this article explains the gameplay role of each item.

## Examples

- Tiles provide the symbols used to build hands.
- Racks help players organize concealed tiles away from other players.

## Common Mistakes

- Using incomplete tile sets.
- Mixing house accessories with required game equipment.
