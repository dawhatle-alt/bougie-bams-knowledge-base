# Game Overview

## Summary

Mahjong is a four-player tile game built around pattern recognition, timing, and table awareness.

## Details

A beginner should first understand the flow: players receive tiles, improve their hand through draws and discards, and work toward a valid winning pattern. In American Mahjong, the current card guides hand selection, but this article stays general and does not reproduce annual card content.

## Examples

- A new player can watch one turn cycle and identify the draw, decision, discard, and reaction window.
- An instructor can use this overview before explaining specific tile groups.

## Common Mistakes

- Trying to memorize every detail before seeing a full turn.
- Assuming all Mahjong styles use the same rules.
