# What Happens If Two Players Call?

## Summary

When more than one player wants a discard, priority rules determine who receives it.

## Details

The exact priority can depend on the situation, especially if Mahjong is involved. Use official guidance for formal play.

## Examples

- A beginner can ask this question before moving tiles.
- A teacher can answer first, then point to the related rule article.

## Common Mistakes

- Relying on a remembered table habit as official guidance.
- Moving tiles before the table agrees on the answer.
