# Can I Call For A Pair?

## Summary

Generally, calling for a pair is restricted and commonly only relevant in specific winning situations; verify with official rules.

## Details

Do not assume every useful discard can be called. The grouping you are completing matters.

## Examples

- A beginner can ask this question before moving tiles.
- A teacher can answer first, then point to the related rule article.

## Common Mistakes

- Relying on a remembered table habit as official guidance.
- Moving tiles before the table agrees on the answer.
