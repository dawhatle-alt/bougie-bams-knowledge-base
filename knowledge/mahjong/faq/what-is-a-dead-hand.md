# What Is A Dead Hand?

## Summary

A dead hand is a hand that can no longer legally continue because of a rule problem or impossible condition.

## Details

Dead-hand questions should be handled calmly and checked against official rules in formal settings.

## Examples

- A beginner can ask this question before moving tiles.
- A teacher can answer first, then point to the related rule article.

## Common Mistakes

- Relying on a remembered table habit as official guidance.
- Moving tiles before the table agrees on the answer.
