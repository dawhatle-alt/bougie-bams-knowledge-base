# Can I Take Back A Discard?

## Summary

No — once the tile touches the table or you have fully named it, the discard is locked. The only exception is a tile that was only partially named and has not yet touched the table.

## Details

A discard becomes final at whichever comes first: the tile touching the tabletop, or the full name being spoken aloud. Saying "Five..." and stopping, with the tile still in hand, leaves room to retract. Saying "Five Bam," or letting the tile land, does not.

## Examples

- "Five Cr—" with the tile still in the air: retractable.
- Tile placed face up silently: locked the moment it touched the table.

## Common Mistakes

- Assuming a silent discard can be recalled because it was never named.
- Letting go of the tile while deciding — placement locks it.
