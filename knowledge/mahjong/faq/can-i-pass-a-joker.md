# Can I Pass A Joker?

## Summary

In many American Mahjong learning contexts, passing a joker during the Charleston is not allowed; consult the current official rules for formal play.

## Details

Treat jokers as special tiles with limits. If a question depends on official procedure, the current NMJL card or rulebook should control.

## Examples

- A beginner can ask this question before moving tiles.
- A teacher can answer first, then point to the related rule article.

## Common Mistakes

- Relying on a remembered table habit as official guidance.
- Moving tiles before the table agrees on the answer.
